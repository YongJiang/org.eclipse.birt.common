JointDataSet Element

Element Overview

Represents a data set joined by serveral data sets on respective join conditions.

Description

The join conditions on which the source data sets joined. Each join condition specify two data sets, which join on this condition, and the type of this join. The left expression, right expression and operator constitute the condition.

Description

Defines the data access attributes of the list of columns in the expected primary result set. This is an optional list of result set column definitions, used only when the underlying data provider cannot provide this information. When contradicting meta-data are found between those provided dynamically by a data provider and those specified in a data set definition, the data providerï¿½s meta-data takes precedence.
