How to set the data type and format of an axis

1  
In Format Chart, navigate to the axis section.
2  
In Type, select a data type from the list, as shown in Figure 16-10.

Figure 16-10  X-Axis section of Format Chart
3  
Choose Format Editor.
The Edit Format dialog box displays options for the data type of the axis. Figure 16-11 shows options for the Date/Time data type.

Figure 16-11  Editing the format of a date
