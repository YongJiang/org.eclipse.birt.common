Tutorial 4:  Creating a stand-alone chart

The tutorial provides step-by-step instructions to build a report that displays order totals organized by product line. The report presents information graphically in a pie chart. In this case, the chart is the entire report.

To create a chart, complete the following tasks:

n  
Set up the report design file and data. Create a new report, data source, and data set.
n  
Add the chart element to the report design. Insert a chart element into a report and select a chart type.
n  
Provide data for the chart. Link the chart to a data set and build the expressions that the chart uses.
n  
Change the chart title.
n  
Refine the chart appearance. Remove the chart legend and modify the labels that identify each sector.
n  
Preview the chart.
