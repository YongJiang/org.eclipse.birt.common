Task 2:  Build a data source

Before you begin designing your report in the layout editor, create a data source to connect your report to the Classic Models database.

1  
Choose Data Explorer.
2  
Right-click Data Sources, and choose New Data Source.
3  
Select Classic Models Inc. Sample Database from the list of data sources, use the default data source name, then choose Next. Connection information about the new data source appears.
4  
Choose Finish. BIRT Report Designer creates a new data source that connects to the sample database.
