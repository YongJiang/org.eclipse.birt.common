How to view and change output columns

1  
Choose Output Columns from the left pane of Edit Data Set. Output Columns displays the names and types of the columns, as shown in Figure 3-6.

Figure 3-6  Viewing the output columns for a data set
2  
To edit the properties of an output column, select the column name, then choose Edit. Edit Output Column displays the properties you can set.
3  
Choose OK to save the edits.
