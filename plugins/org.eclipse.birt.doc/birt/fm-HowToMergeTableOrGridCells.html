How to merge table or grid cells

1  
Select the cells to merge by using Shift-click. A border appears around the selected cells, as shown in Figure 7-18.

Figure 7-18  Cells selected for merging
2  
Right-click within the border, then choose Merge Cells from the context menu. The cells merge into one, as shown in Figure 7-19.

Figure 7-19  Merged cells
