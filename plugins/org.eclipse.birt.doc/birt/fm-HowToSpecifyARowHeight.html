How to specify a row height

1  
Select the tab at the bottom left corner of the grid or table. Guide cells appear at the top and left sides of the grid or table.
2  
Select the guide cell of the row to resize.
3  
In Property Editor, under Properties, choose General. Property Editor displays the general properties of the row, as shown in Figure 7-16.

Figure 7-16  Property Editor showing row properties
4  
Specify a value for Height. You can specify different units of measurements, including inches, centimeters, millimeters, and points.
